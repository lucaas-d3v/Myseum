# Myseum

Basicamente, uma API para catalogação e preservação de jogos.

# Requisitos/Stack (v0.0.1)

* Python 3.11.2+
* MariaDB >= 10.11 | MySQL >= 8.0

# Rodando

# 1. Criar o ambiente virtual
```bash
python -m venv venv
```

# 2. Ativar o ambiente (Escolha conforme seu OS/Terminal)
```bash
source venv/bin/activate      # Linux / macOS / Git Bash
.\venv\Scripts\Activate.ps1   # Windows (PowerShell)
.\venv\Scripts\activate.bat   # Windows (CMD)
```

# 3. Atualizar o gerenciador de pacotes
```bash
pip install --upgrade pip
```
